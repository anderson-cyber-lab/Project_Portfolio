# Sprint 15: CompTIA Security+ Certification

## Overview

Completed the CompTIA Security+ certification as the capstone milestone of the TripleTen Cybersecurity program. Security+ is a globally recognized, vendor-neutral certification validating foundational and intermediate cybersecurity skills across a broad range of domains, and is compliant with DoD 8570 requirements for information assurance roles.

## Domains Covered

The Security+ exam covers the following core domains:

- **General Security Concepts** — Security controls, cryptography fundamentals, authentication, and authorization
- **Threats, Vulnerabilities & Mitigations** — Malware types, social engineering, vulnerability scanning, and threat intelligence
- **Security Architecture** — Network segmentation, cloud security, infrastructure hardening, and zero trust
- **Security Operations** — Incident response, log analysis, SIEM usage, identity management, and endpoint security
- **Security Program Management & Oversight** — Risk management, compliance frameworks, data privacy, and security policies

## Skills Validated

- Identifying and mitigating common attack vectors (phishing, brute force, ransomware, insider threats)
- Implementing and evaluating security controls across network, endpoint, and cloud environments
- Applying cryptographic principles to protect data in transit and at rest
- Performing vulnerability assessments and interpreting scanner output
- Executing incident response procedures and documenting findings
- Understanding compliance frameworks (NIST, ISO 27001, SOC 2, HIPAA, PCI-DSS)
- Applying zero trust architecture and least-privilege principles

## Relevance to Portfolio

This certification validates the theoretical and conceptual foundations underlying every hands-on project completed throughout the TripleTen program — from network scanning and penetration testing (Sprints 2–9) to SIEM deployment and advanced threat hunting (Sprints 11–14). It demonstrates readiness for SOC Analyst, Security Analyst, and Junior Penetration Tester roles.
